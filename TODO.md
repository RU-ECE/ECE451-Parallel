# Getting Started

1. **Get set up for GitHub (you can
   follow [this](https://docs.google.com/document/d/1WOJqA5Vdo9QbeYMGn8pN1B8Uo8-1EnfZAMCAdkDKjFk/edit) guide)**
2. **Create your repo with the correct directory structure**
	- Make sure your repository layout matches the assignment / course expectations.
	- Commit and push an initial version so you know everything works.
3. **Make sure you can log into Amarel**
	- Use the instructions in the root directory: [`amarel_instructions.md`](amarel_instructions.md)
	- Verify:
		- You can connect via SSH.
		- You can transfer files.
		- You can submit a simple test job if required.
